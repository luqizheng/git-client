⚙️ 技术实现架构

Store 设计：

// stores/rightPanel.ts
interface RightPanelState {
  visible: boolean
  mode: 'commit' | 'staging' | null
  selectedCommitSha: string | null
  width: number  // default: 320, min: 240, max: 480
}

组件结构：

RightPanel.vue
├── CommitDetails.vue       // mode === 'commit'
│   ├── CommitHeader.vue     // sha, author, date, parents
│   └── ChangedFilesList.vue // file list with diff link
└── StagingPanel.vue         // mode === 'staging'
    ├── FileStatsHeader.vue   // "N file changes on branch"
    ├── UnstagedFiles.vue     // file list + stage button
    ├── StagedFiles.vue       // file list + unstage button
    └── CommitEditor.vue      // summary, description, options

事件触发：

  • 点击 Graph 中的 commit 节点 → store.showPanel('commit', sha)
  • 点击左侧导航的 "Working Files" 图标 → store.showPanel('staging')
  • 点击关闭按钮或 Escape → store.hidePanel()
  • 拖拽分隔线 → 更新 store.width
