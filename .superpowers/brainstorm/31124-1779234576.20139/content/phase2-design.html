2.4 Undo/Redo + Discard

↩ Undo / ↪ Redo
实现方式：Git reflog 操作栈
• 每次操作（commit/merge/rebase/reset）记入栈
• Undo = git reset 到 reflog 前一条
• Redo = 前进到 reflog 后一条
• 栈深度默认 50 条
• 快捷键 Ctrl+Z / Ctrl+Shift+Z
🗑 Discard 更改
单个文件：git checkout -- <file>
全部未暂存：git checkout -- .
确认对话框："确定丢弃 X 个文件的更改？"
新增文件：git clean -f（需单独确认）
• 右键菜单 → Discard This File
• StagingPanel 顶部 → Discard All 按钮
