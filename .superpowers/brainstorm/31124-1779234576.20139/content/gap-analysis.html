GitKraken vs 我们 — 功能对比与差距分析

对标 GitKraken 免费版核心功能，红色为缺失/未完成，绿色为已完成

🔴 P0 — 必须完成（核心 Git 操作缺失）

!

Diff 视图（Monaco Diff）

diff/ 目录完全为空。Commit详情中只能看文件列表，无法查看具体代码差异。GitKraken 核心体验。

!

交互式 Rebase

RebaseDialog 仅有壳，无法拖拽排序 commit、squash、reword。GitKraken 拖拽式 rebase 是杀手功能。

!

Cherry-Pick

右键菜单无 Cherry-Pick 选项。GitKraken 右键 commit 即可 cherry-pick。

!

Undo / Redo

工具栏 Undo/Redo 按钮始终 disabled。GitKraken 一键撤销/重做 Git 操作。

!

Discard 丢弃更改

StagingPanel 中 discard 功能仅 toast 提示"不支持"。无法丢弃单个文件或全部未暂存更改。

!

内置代码编辑器

无法在 diff 视图中直接编辑文件解决冲突。GitKraken 内置编辑器可原地修改。

🔴 P0 — 冲突解决（核心体验缺失）

!

三方合并冲突解决

ConflictResolver 仅 3 个按钮（Ours/Theirs/Base），无真正的三栏 diff 对比。ThreeWayDiff 组件为空壳。GitKraken 有完整的 3 栏可视化冲突解决。

!

合并冲突预测提示

无 Predictive Merge Conflict Alerts。GitKraken 会在合并前预测冲突数量。

🟡 P1 — 重要但非紧急

~

File History 文件历史

无法查看单个文件的完整修改历史。GitKraken 右键文件可查看 File History。

~

Command Palette 命令面板

无 Ctrl+Shift+P 快捷命令面板。GitKraken 的 Command Palette 可快速执行任何操作。

~

Git LFS 支持

无 Git LFS 管理。大文件仓库必需。

~

.gitignore 可视化编辑

无 .gitignore 管理。GitKraken 可视化管理忽略规则。

~

GPG 签名提交

GPG Key 管理界面有，但 CommitEditor 无 GPG 签名选项。GitKraken 支持 GPG 签名。

~

拖拽式 Push/Pull

无法拖拽分支到远程推送。GitKraken 可拖拽分支到 remote 推送。
