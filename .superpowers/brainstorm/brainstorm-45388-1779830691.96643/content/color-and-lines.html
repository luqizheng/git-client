颜色分配规则

  1. 新列创建时取 palette[column % 8]
  2. Replace（branch child 继续使用父列）→ 继承父列颜色
  3. 回收：active branch 结束 → 列释放，列号收缩，后续新列复用
  4. 所有 SVG 线/节点/标签使用同一色值 → 保持视觉一致性

暗色/亮色双主题适配

亮色
用 60% 饱和度变体，线条加透明
暗色
用 100% 饱和度变体，线条加透明

实现方式：用 rgba(color, alpha) 作为线段颜色，节点用纯色。

线段生成 — 连线规则

基于列分配结果绘制 SVG 线段

三种连线类型

场景当前改进
同列→同列垂直连线不变
同列→不同列水平+垂直水平+垂直（直角，不变）
不同列→同列水平+垂直水平+垂直（存在多个父列时的合并节点）

新增规则：合并提交如果有两个父提交在不同列，画两条入线

线段生成算法

for each commit c (oldest to newest):
  for each parent p of c:
    if c.column == p.column:
      // 垂直连线 (单单线)
      emit vertical(c.y, p.y, c.column, c.color)
    else:
      // 直角连线 (L形)
      // 从 p 所在列右端 draw 到 c 所在列左端
      emit horizontal(p.x + RADIUS, p.y, c.x - RADIUS, p.y)
      emit vertical(p.y, c.y, c.column, c.color)
