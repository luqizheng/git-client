A. 左右分栏（列表 + 详情）

左侧 commit 列表 + 搜索栏，右侧选中 commit 的详情和文件变更。类似 GitKraken。

B. 上下分区（列表 + 底部详情）

上方 commit 列表 + 搜索，下方展开选中 commit 的摘要。底部面板可拖拽调整高度。
